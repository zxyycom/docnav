# Docnav v0.1.0-beta.1

`v0.1.0-beta.1` 是 Docnav 的首个 Beta 版本，范围是当前已实现并验证的 CLI 与内置 Markdown adapter。主要导航流程是先用 `outline` 获取结构和 ref，再用 `read --ref` 有限、可继续地读取对应内容；本次发布收尾不新增运行时行为。

发布候选已通过 full workspace verification、文档与 OpenSpec validation、core CLI smoke，以及 Linux 和 Windows canonical package 的 build、verify 与 smoke。

## 下载

在此 prerelease 的 **Assets** 区域按目标平台下载 binary 和同名 checksum：

| Target | Binary | Checksum |
| --- | --- | --- |
| Linux `x86_64-unknown-linux-gnu` | `docnav-v0.1.0-beta.1-x86_64-unknown-linux-gnu` | `docnav-v0.1.0-beta.1-x86_64-unknown-linux-gnu.sha256` |
| Windows `x86_64-pc-windows-msvc` | `docnav-v0.1.0-beta.1-x86_64-pc-windows-msvc.exe` | `docnav-v0.1.0-beta.1-x86_64-pc-windows-msvc.exe.sha256` |

## 校验与启动

Linux：

```bash
sha256sum --check docnav-v0.1.0-beta.1-x86_64-unknown-linux-gnu.sha256
chmod +x docnav-v0.1.0-beta.1-x86_64-unknown-linux-gnu
./docnav-v0.1.0-beta.1-x86_64-unknown-linux-gnu version
./docnav-v0.1.0-beta.1-x86_64-unknown-linux-gnu --help
```

Windows PowerShell：

```powershell
$asset = "docnav-v0.1.0-beta.1-x86_64-pc-windows-msvc.exe"
$expected = (((Get-Content "$asset.sha256" -Raw).Trim() -split "\s+")[0]).ToLowerInvariant()
$actual = (Get-FileHash $asset -Algorithm SHA256).Hash.ToLowerInvariant()
if ($actual -ne $expected) { throw "SHA-256 mismatch" }
& ".\$asset" version
& ".\$asset" --help
```

`version` 应输出 `docnav 0.1.0-beta.1`。

## 已知限制

- 当前 Beta 只提供上述两个 x86_64 target；不包含 macOS、Arm 或其它 target。
- Assets 只包含两个裸 binary 和两个 `.sha256`，不提供归档包或 installer。
- SHA-256 用于完整性校验；当前不提供 binary signature 或平台公证。
- Beta 范围以当前已验证的 CLI 和内置 Markdown adapter 为准，不包含尚未实现的计划能力。

## 发布与修复边界

公开后的 asset 集合、checksum、binary version 或基本执行检查如果失败，此版本不会被记录为完成交付，且既有 assets 不会被原位替换。修复将使用递增的 prerelease version，重新经过完整 baseline、打包、promotion 和发布复核流程。
